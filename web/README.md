# Electric Calculator - Web Version

Веб-версия калькулятора электрических параметров для быстрых расчётов.

## Возможности

- **Расчёт мощности (P)** по напряжению и току
- **Расчёт тока (I)** по напряжению и мощности
- **Расчёт напряжения (U)** по току и мощности
- **Расчёт сечения провода** по току:
  - Минимальное сечение (по формуле)
  - Диаметр провода
  - Рекомендуемое стандартное сечение
- Переключение **1 фаза / 3 фазы**
- Выбор материала провода: **Алюминий (Al) / Медь (Cu)**
- Настройка коэффициента мощности **cos φ** (0.1 - 1.0)
- Пресет напряжения 380В для 3-фазной сети
- Сохранение настроек в браузере
- Метрическая/имперская система (mm²/inch²)

## Запуск

### Вариант 1: Просто открыть файл
Просто откройте файл `index.html` в любом современном браузере:
- На компьютере: дважды кликните по файлу
- На телефоне: откройте через файловый менеджер

### Вариант 2: Локальный сервер (рекомендуется)
```bash
# Если установлен Python 3
cd web
python3 -m http.server 8000

# Если установлен Node.js
npx serve web
```

Затем откройте в браузере: `http://localhost:8000`

### Вариант 3: GitHub Pages
Загрузите файлы в репозиторий GitHub и включите GitHub Pages.

## Использование на iPhone/iPad

1. Откройте страницу в Safari

3. Выберите "На экран «Домой»"
4. Приложение появится на главном экране как нативное

## Формулы расчёта

### Для 1-фазной сети:
```
P = U × I × cosφ
I = P / (U × cosφ)
```

### Для 3-фазной сети:
```
P = √3 × U × I × cosφ
I = P / (√3 × U × cosφ)
```

### Расчёт сечения провода:
**Для меди:**
```
S = 0.01 × I^1.64
```

**Для алюминия:**
```
S = 0.0165591 × I^1.575
```

Где:
- **P** — мощность (Вт)
- **U** — напряжение (В)
- **I** — ток (А)
- **cosφ** — коэффициент мощности
- **S** — сечение провода (мм²)

### Рекомендуемые сечения:
Приложение автоматически подбирает стандартное сечение провода по таблице в зависимости от материала и тока.
